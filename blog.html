What is the Document Object Model (DOM)?

When we render an HTML file in the browser, the browser processes the HTML document, and each node in the HTML file is represented as an object. This structure forms a tree, which is known as the DOM (Document Object Model).

How do you select an element from the DOM?

document.getElementById('id')

What is the event delegation in the context of the DOM, and why is it useful?

Event delegation is essentially a concept where if an event is triggered on a parent element, it automatically affects its child elements. This can also work in reverse; if an event is triggered on a child element, the parent element can be affected by that event as well. Event delegation is important because it allows for efficient event handling, improving the performance of a website by managing events more effectively. This is why event delegation is useful.

How do you manipulate an element's attributes and styles using the DOM?

let elem = document.getElementById('id') elem.style.backgroundColor = 'red';
document.getElementById("id").setAttribute("src", "image.jpg");
